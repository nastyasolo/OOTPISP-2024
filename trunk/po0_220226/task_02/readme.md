### Конструктор базового класса ###

```c++

Transport_Vehicle::Transport_Vehicle(const int maxSpeed) : mmaxSpeed(maxSpeed) {
    this->Add();
}

void Transport_Vehicle::Add() {
    if (begin == nullptr) {
        begin = this;
    }
    next = nullptr;
    if (last != nullptr) {
        last->SetNext(this);
    }
    last = this;
    last->SetNext(nullptr);
}

```

### Статические компоненты класса ###

```c++

    static Transport_Vehicle* begin;
	static Transport_Vehicle* last;

```

### Реализация метода для просмотра списка ###

```c++

void Transport_Vehicle::Print() {
    Transport_Vehicle* ptr = begin;

    while (ptr != nullptr) {
        ptr->Show();
        ptr = ptr->next;
    }
}

```

### Реализация метода для добавления объекта в список ###

```c++

void Transport_Vehicle::Add() {
```
